📋 ЧТО НУЖНО:
   • Сервер VPS (DigitalOcean, Vultr, Aeza) - $5-6/мес

   • Пароль от сервера

⚡ ШАГ 1: Арендовать сервер (5 минут)
   DigitalOcean: https://digitalocean.com
   Тариф: $6/мес (Droplet 512MB)
   ОС: Ubuntu 22.04

⚡ ШАГ 2: Скопировать проект на сервер (2 минуты)
   cd /Users/alexgreek/Desktop/new-project
   scp -r . root@ВАШ_IP:/root/claude-trading-bot

⚡ ШАГ 3: Подключиться к серверу (1 минута)
   ssh root@ВАШ_IP

⚡ ШАГ 4: Установить Node.js и зависимости (5 минут)
   apt-get update
   curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
   apt-get install -y nodejs npm
   
   cd /root/claude-trading-bot
   npm install

⚡ ШАГ 5: Установить PM2 и запустить бота (2 минуты)
   npm install -g pm2
   pm2 start src/index.js --name trading-bot
   pm2 save
   pm2 startup

   👆 Скопируйте команду которую покажет pm2 startup!
