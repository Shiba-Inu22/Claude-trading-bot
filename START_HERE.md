## ❓ Troubleshooting:

### Бот не отвечает:
- Проверьте что API ключ правильный
- Убедитесь что добавили карту в Anthropic
- Проверьте баланс в Anthropic dashboard

### Ошибка "Invalid API key":
- Перепроверьте ключ в .env
- Убедитесь нет пробелов в начале/конце

### Ошибка "Insufficient credits":
- Добавьте средства в Anthropic billing
