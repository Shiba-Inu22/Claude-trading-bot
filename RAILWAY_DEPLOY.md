## ⚠️ ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### Бот не запускается:

**Проверьте логи:**
```
Railway Dashboard → Logs
```

**Ошибка "Cannot find module":**
Убедитесь что `package.json` содержит зависимости:
```json
{
  "dependencies": {
    "node-telegram-bot-api": "^0.x.x",
    "axios": "^1.x.x",
    "dotenv": "^16.x.x"
  }
}
```

### Бот отвечает медленно:
- Проверьте логи на ошибки
- Убедитесь что API ключи правильные

### Проект "засыпает":
На бесплатном тарифе Railway может останавливать проект при простое. 

**Решение:**
1. Используйте UptimeRobot (бесплатно)
2. Настройте пинг каждые 5 минут
