# 🔑 Как получить Anthropic (Claude) API ключ

## Шаг 1: Создайте аккаунт на Anthropic

1. Перейдите на https://console.anthropic.com
2. Нажмите "Sign Up" или "Log In"
3. Войдите через Google или создайте аккаунт

## Шаг 2: Добавьте способ оплаты

Claude API платный (но недорогой):
- Claude Sonnet: ~$0.003 за 1K токенов
- Claude Opus: ~$0.015 за 1К токенов

**Важно:** Без добавления карты API не будет работать!

### Как добавить карту:
1. Зайдите в https://console.anthropic.com/settings/billing
2. Нажмите "Add Payment Method"
3. Введите данные карты
4. Подтвердите

## Шаг 3: Создайте API ключ

1. Перейдите на https://console.anthropic.com/account/keys
2. Нажмите **"Create Key"**
3. Дайте название (например: `ComputerAgent`)
4. Скопируйте ключ (покажут только один раз!)

Ключ выглядит так: `sk-ant-api03-xxxxxxxxxxxxx`

## Шаг 4: Настройте .env файл

Откройте файл `.env` в папке проекта и замените:

```bash
ANTHROPIC_API_KEY=sk-ant-api03-ВАШ_КЛЮЧ
TELEGRAM_BOT_TOKEN=8166473987:AAFE3DrNIcd_n39koz-Y70Mq-n96p2N8Vsw
TELEGRAM_CHAT_ID=226166473
```

## Шаг 5: Проверьте работу

Запустите бота:
```bash
npm start
```

Если всё правильно - бот отправит вам сообщение в Telegram!

## ❓ Troubleshooting

### Ошибка "Invalid API key":
- Проверьте что скопировали весь ключ без пробелов
- Убедитесь что добавили карту в billing settings

### Ошибка "Rate limit":
- Вы превысили лимит запросов
- Подождите немного или upgrade тариф

### Ошибка "Insufficient credits":
- Добавьте средства на счет
- Проверьте billing dashboard
